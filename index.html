¿Qué incluye cada casa?

Cada modelo se entrega según el kit que elijas.
Revisa las diferencias y elige el mejor para ti.

1. KIT BÁSICO

  • Paneles madera 2x3 (Medialuna/Tinglado).
  • Cerchas 1x4 con elevación 1,20 m.
  • Fieltro asfáltico y zinc.
  • Caballetes (sin limahoyas).
Ideal para construir por etapas.

2. KIT INTERMEDIO

  • Todo lo del Kit Básico.
  • Ventanas de madera con marco.
  • Puertas interiores terciado.
  • Yeso cartón 10mm (cielo y 1 cara).
Para entrega más rápida.

3. SUPER KIT

  • Todo lo anterior + Puertas pino ext.
  • Yeso cartón ambas caras.
  • Piso completo (pilotes, vigas, terciado).
Lista para instalación inmediata.
